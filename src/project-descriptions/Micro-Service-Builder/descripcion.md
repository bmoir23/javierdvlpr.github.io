# Rapid Micro-Service Builder
Herramienta enfocada en desarrolladores que utiliza la orquestación de n8n e interfaces personalizadas de Next.js para reducir los tiempos de despliegue de servicios en producción de semanas a horas.

## Stack
Next.js, React, n8n, Tailwind CSS, Node.js, Orquestación de APIs, REST APIs
