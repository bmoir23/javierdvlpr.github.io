# Marketing Automation System
Desarrollo de una plataforma modular de generación de leads que cuenta con integraciones validadas de Airtable, flujos de trabajo de correo electrónico automatizados y orquestación de CRM con n8n, diseñada para un despliegue rápido y monetización.

## Stack
n8n, React, integración Airtable, flujos de correo automatizados, orquestación de CRM, HubSpot, Node.js
