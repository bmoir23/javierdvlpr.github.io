# Syncc Executive Agent OS
Arquitectura de una plataforma en la nube multiinquilino donde cada URL individual levanta dinámicamente un agente de IA dedicado, completo con su propio entorno de ejecución, protocolos de comunicación de agente a agente, memoria vectorial y APIs para desarrolladores.

## Stack
Cloudflare Workers, Next.js, LangChain, Model Context Protocol (MCP), Bases de datos vectoriales, Búsqueda semántica, Serverless
