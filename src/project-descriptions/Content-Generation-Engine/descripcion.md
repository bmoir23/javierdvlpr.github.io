# Content Generation Engine
Diseño de un pipeline automatizado utilizando modelos de IA y n8n para extraer temas en tendencia, clasificarlos mediante agentes de puntuación automáticos y publicar contenido multiplataforma con seguimiento de interacción para optimización continua.

## Stack
Python, n8n, Transformers, Hugging Face, agentes de puntuación automática, pipelines de extracción, PostgreSQL
